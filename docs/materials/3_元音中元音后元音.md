# Day 03｜元音 中元音后元音

> 今日目标
- 掌握中元音：/ə/ /ɜː/
- 掌握后元音：/ɑː/ /ɒ/ /ɔː/ /ʊ/ /uː/
- 理解并练习圆唇程度的递进

## 舌位/腔体（或规则讲解）
- **中元音发音要点**：
  - /ə/（schwa）：最放松的音，舌头自然中央位置，常用于弱读
  - /ɜː/：长中元音，舌位中央，嘴唇微开
- **后元音发音要点**：
  - /ɑː/：低后元音，口型最大，不圆唇
  - /ɒ/：低后元音，轻微圆唇（英式）
  - /ɔː/：中后元音，中等圆唇
  - /ʊ/：高后元音，圆唇，短音
  - /uː/：最高后元音，圆唇，长音
- **常见错误与纠正方法**：
  - 错误：/ʊ/和/uː/不分
  - 纠正：注意长短和紧张度差异

## 单词演示（高频词与最小对立对）
- /ə/: about, sofa, banana, problem, national
- /ɜː/: bird, work, turn, learn, earth, nurse
- /ɑː/: car, father, start, heart, large
- /ɒ/: hot, box, stop, watch, want (英式)
- /ɔː/: call, law, autumn, water, thought
- /ʊ/: book, put, good, would, could
- /uː/: food, moon, soon, blue, true
- 最小对立对：
  - full /fʊl/ vs fool /fuːl/
  - cot /kɒt/ vs caught /kɔːt/
