# 以-s结尾的单词发音规则

## 发音规则
当单词以**清辅音** (/p/, /t/, /k/, /f/, /θ/) 结尾时,加-s发音为 **/s/**

## 单词示例

| 单词 | 音标 | 说明 |
|------|------|------|
| cats | /kæts/ | 以/t/结尾 |
| books | /bʊks/ | 以/k/结尾 |
| cliffs | /klɪfs/ | 以/f/结尾 |
| hats | /hæts/ | 以/t/结尾 |
| cups | /kʌps/ | 以/p/结尾 |
| laughs | /læfs/ | 以/f/结尾 |
| thoughts | /θɔːts/ | 以/t/结尾 |
| kicks | /kɪks/ | 以/k/结尾 |
| packs | /pæks/ | 以/k/结尾 |
| shifts | /ʃɪfts/ | 以/t/结尾 |

## 总结
这些单词的共同特点是:词尾的清辅音后加-s时,**-s读作清辅音/s/**,而不是浊辅音/z/。
