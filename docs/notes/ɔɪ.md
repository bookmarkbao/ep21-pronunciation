## 双元音 /ɔɪ/

### 单词练习

boy, point, voice, join, enjoy, choice, destroy, lawyer, boyfriend, poison, noise, avoid, oil, joy, toilet, disappoint, royal, appointment, toy, annoy

---

## 🎯 /ɔɪ/ 发音要领

### **1. 基本特征**
- **类型：** 双元音（diphthong）
- **组成：** /ɔ/ → /ɪ/（从 /ɔ/ 滑向 /ɪ/）
- **音长：** 长元音
- **发音过程：** 动态滑动，从圆唇到展唇

---

### **2. 发音步骤**

| 步骤 | 动作 | 说明 |
|------|------|------|
| **起始音 /ɔ/** | 嘴唇圆形，舌位中低后 | 类似中文"奥"但更圆 |
| **滑动过程** | 嘴唇从圆变扁，舌位前移上升 | 快速过渡 |
| **结束音 /ɪ/** | 嘴唇展开，舌位高前 | 类似中文"衣"但更轻 |

---

### **3. 发音要点**

✅ **正确发音：**
- 起始嘴唇圆形突出
- 舌位从后低到前高
- 嘴唇从圆唇到展唇
- **圆唇→展唇变化明显（关键！）**
- 滑动连贯有力
- 整体音色饱满

❌ **常见错误：**
- ❌ 发成单一的 /ɔ/ 音（没有滑动）
- ❌ 起始音不够圆（缺乏圆唇）
- ❌ 发成 /aɪ/ 音（起始音错误）
- ❌ 滑动不明显
- ❌ 结尾太重（应该轻收）
- ❌ 嘴唇变化不够

---

### **4. 口型变化图**

```
起始 /ɔ/          滑动中          结束 /ɪ/
   😮      →      😐      →      😊
 (圆唇)       (渐展开)       (展唇)
 嘴唇圆         嘴唇扁         嘴唇平
```

---

### **5. 舌位变化**

```
高 ←── 舌位前移上升 ←──┐
                      │ /ɪ/ 结束
中低 ←── /ɔ/ 起始 ─────┘
后                   前
    (舌位从后到前)
```

---

### **6. 嘴唇动作（最重要！）**

```
  正面观察：
  
  /ɔ/: 😮 (圆形，突出)
       ↓
  /ɪ/: 😊 (展开，扁平)
```

## 🎵 发音练习技巧

### **1. 慢动作练习**
- 第一步：发 /ɔ/（嘴唇圆形突出）- **保持 1 秒**
- 第二步：慢慢滑向 /ɪ/（嘴唇展开）- **保持 1 秒**
- 第三步：连贯发音 /ɔɪ/ - **流畅有力**

---

```
照镜子观察：

3. 结束时嘴唇是否变平？ ✅
4. 整个过程连贯吗？ ✅
```

## 📊 /ɔɪ/ 音的特点总结

| 特征 | 说明 |
|------|------|
| **音标类型** | 双元音（由两个音组成） |
| **起始音** | /ɔ/（中低后圆唇元音） |
| **结束音** | /ɪ/（高前展唇元音，轻读） |
| **滑动方向** | 从后到前，从圆到扁 |
| **音长** | 长元音（比单元音长） |
| **最常见拼写** | oi（词中）、oy（词尾） |
| **使用频率** | 中等（比 /eɪ/ /oʊ/ 少见） |

## ⚠️ 特别注意事项

### **1. 不要发成单音**
- ❌ 错误：boy /bɔ/（只发 /ɔ/）
- ✅ 正确：boy /bɔɪ/（要滑向 /ɪ/）

### **2. 不要混淆 /ɔɪ/ 和 /aɪ/**
- /ɔɪ/：boy, toy（圆唇开始）
- /aɪ/：buy, tie（大开口开始）

### **3. oi 和 oy 位置规律**
- **oi** 在词中：point, voice, join
- **oy** 在词尾：boy, toy, enjoy
- **例外很少！** 几乎100%准确

### **4. 嘴唇变化是关键**
- 起始必须圆唇
- 结束必须展唇
- 中间平滑过渡
