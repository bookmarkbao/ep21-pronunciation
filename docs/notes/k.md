# /k/ 音标练习单词列表

can, like, come, think, look, take, okay, back, make, thank, could, talk, call, work, because, ask, kill, keep, kid, kind

---

## 🎯 /k/ 发音说明

### **发音方法：**
1. **舌位：** 舌后部抬起，抵住软腭（口腔后上方）
2. **嘴型：** 嘴巴自然张开
3. **气流：** 气流在口腔后部积聚，舌后部突然离开软腭，气流爆破而出
4. **声带：** **不震动**（清辅音）
5. **口诀：** 舌后部抵住软腭，发出"咔"的爆破声

---

## 📍 /k/ 拼写规律

| 拼写形式 | 例词 |
|----------|------|
| **c-** (词首，后接 a/o/u) | **c**an, **c**ome, **c**ould, **c**all |
| **k-** (词首，后接 i/e) | **k**ill, **k**eep, **k**id, **k**ind |
| **-k** (词尾) | thin**k**, loo**k**, ta**k**e, oka**y**, bac**k**, ma**k**e, than**k**, tal**k**, wor**k**, as**k** |
| **-ck** (词尾) | ba**ck** |
| **-c-** (词中) | be**c**ause |
| **li-ke** (特殊) | li**ke** |

## 🔄 常见对比

| 音标 | 特点 | 例词 | 提示 |
|------|------|------|------|
| **/k/** | 清辅音，不震动，爆破音 | **c**an, ba**ck** | 舌后部抵软腭，声带不震动 |
| **/g/** | 浊辅音，震动，爆破音 | **g**o, ba**g** | 舌后部抵软腭，声带震动 |
| **/t/** | 清辅音，齿龈爆破音 | **t**an | 舌尖抵齿龈 |
| **/h/** | 清辅音，喉擦音 | **h**and | 气流从喉咙出，不受阻 |

**对比练习：** 
- /k/ vs /g/: came → game, class → glass, back → bag, lock → log
- /k/ vs /t/: came → tame, care → tear, back → bat
- /k/ vs /h/: came → ham, kind → hind

## 🗣️ 练习技巧

✅ **记忆口诀：** 舌后部"咔"的爆破声 = /k/  
✅ **检查方法：** 
   - 把手放在嘴前，能感觉到**强烈的气流**喷出
   - 声带**不震动**（把手放喉咙上检查）
   - 在纸条或蜡烛前发音，能看到纸条/火焰晃动
   - 舌后部能感觉到抵住软腭
✅ **常见错误：** 
   - ❌ 声带震动（发成 /g/）
   - ❌ 舌尖抵齿龈（发成 /t/）
   - ❌ 气流太弱（爆破不明显）
   - ❌ 舌位太靠前

## 💡 重要提示

⭐ **/k/** 是**软腭爆破音**，是 /g/ 的清音对应  
⭐ **关键特征：** 舌后部抵住软腭，气流积聚后爆破，声带不震动  
⭐ **拼写规律：**
   - **c** + a/o/u 发 /k/：**c**an, **c**ome, **c**ould, **c**all
   - **c** + i/e 通常发 /s/：city, center（不在本列表中）
   - **k** + i/e 发 /k/：**k**ill, **k**eep, **k**id, **k**ind
   - 词尾 **-k** 或 **-ck** 都发 /k/：thin**k**, bac**k**, tal**k**, wor**k**
   - **-ck** 只出现在短元音后：ba**ck**, clo**ck**, lu**ck**
   - **-ke** 通常在长元音或双元音后：li**ke**, ta**ke**, ma**ke**
⭐ **送气特点：**
   - 词首的 /k/ **送气明显**：**c**an, **c**ome, **k**eep（气流很强）
   - 在 /s/ 后的 /k/ **不送气**：s**k**y, s**c**hool（气流较弱）
   - 词尾的 /k/ **轻收音**：thin**k**, loo**k**, bac**k**（爆破不完全）
⭐ **特殊情况：**
   - **talk** /tɔːk/, **walk** /wɔːk/ 中的 l 不发音
   - **could** /kʊd/ 中的 l 不发音
   - **okay** /əʊˈkeɪ/ 拼写特殊

## 📌 拼写规律详解

### **字母 c 的发音规则：**

| 后接字母 | 发音 | 例词 |
|---------|------|------|
| a, o, u | /k/ | **c**an, **c**ome, **c**ould, **c**all |
| i, e, y | /s/ | city, center, cycle |

### **字母 k 的发音：**
- **k** 几乎总是发 /k/ 音：**k**ill, **k**eep, **k**id, **k**ind
- 极少数例外：**know, knife, knee** 中的 k 不发音

### **词尾 -k vs -ck：**

| 拼写 | 条件 | 例词 |
|-----|------|------|
| **-ck** | 短元音后 | ba**ck**, clo**ck**, lu**ck**, si**ck** |
| **-k** | 长元音/双元音/辅音后 | thin**k**, loo**k**, tal**k**, wor**k**, as**k** |
| **-ke** | 长元音/双元音后 | li**ke**, ta**ke**, ma**ke** |

## 💡 记忆技巧

**C vs K 选择规则：**
- **看后面的元音！**
  - a, o, u → 用 **c**：can, come, could
  - i, e → 用 **k**：kill, keep, kid, kind

**词尾拼写规则：**
- **短促的音** → 用 **-ck**：back
- **长长的音 / 多音节** → 用 **-k**：think, look, talk, work
