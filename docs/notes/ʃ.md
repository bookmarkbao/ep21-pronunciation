## 🎯 /ʃ/ 发音说明

### **发音方法：**
1. **舌位：** 舌尖接近上齿龈后部，但不接触，舌面抬起
2. **嘴型：** 嘴唇略微向前突出，呈圆形（像要说"xu"）
3. **气流：** 气流从舌面和硬腭之间的缝隙通过，形成摩擦音
4. **声带：** **不震动**（清辅音）
5. **口诀：** 像中文"嘘"的声母，或者让别人安静时的"shh"

## 📍 /ʃ/ 拼写规律

| 拼写形式 | 例词 |
|----------|------|
| **sh-** (词首/词尾) | **sh**e, **sh**ould, **sh**ow, **sh**ut, **sh**ot, **sh**all, **sh**oot, wi**sh**, fi**sh**, pu**sh**, fini**sh** |
| **-ti-** (后接on/al) | sta**ti**on, informa**ti**on |
| **-ci-** (后接al/ous) | spe**ci**al |
| **-s-** (后接ure) | **s**ure |
| **-ss-** (后接ion) | 某些词中 |

---

## 🔄 常见对比

| 音标 | 特点 | 例词 | 提示 |
|------|------|------|------|
| **/s/** | 舌尖音 | **s**it, bu**s** | 舌尖在上齿龈 |
| **/ʃ/** | 舌面音 | **sh**ip, pu**sh** | 舌面抬起，嘴唇圆 |

**对比练习：** sea → she, sock → shock, save → shave

## 🗣️ 练习技巧

✅ **记忆口诀：** "嘘！安静！" = /ʃ/  
✅ **嘴型提示：** 想象吹灭蜡烛的嘴型  
✅ **常见错误：** ❌ 发成 /s/（嘴型不够圆，舌位太靠前）
