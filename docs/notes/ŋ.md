# /ŋ/ 音标深度分析

---

## 🎯 /ŋ/ 音的特点

| 特征 | 说明 |
|------|------|
| **发音位置** | 舌根顶住软腭（口腔后部） |
| **发音方式** | 鼻音（气流从鼻腔出） |
| **声带** | 振动（浊音） |
| **关键** | 舌尖**不碰**上齿龈！ |

---

## 📊 按发音模式分组

### **1️⃣ -ing 组（/ɪŋ/ 结尾）**

| 单词 | 音标 | 特点 |
|------|------|------|
| **thing** | /θɪŋ/ | 短元音 /ɪ/ + /ŋ/ |
| **something** | /ˈsʌmθɪŋ/ | 复合词，重音在首 |
| **ring** | /rɪŋ/ | r 音后接 /ɪŋ/ |
| **bring** | /brɪŋ/ | 辅音群 br + /ɪŋ/ |
| **sing** | /sɪŋ/ | 简单清晰的 /ɪŋ/ |
| **king** | /kɪŋ/ | k 爆破后接 /ɪŋ/ |
| **morning** | /ˈmɔːrnɪŋ/ | 长元音 /ɔːr/ + n + /ɪŋ/ |

**发音要点：**
- /ɪ/ 是短促的"衣"音
- 然后舌根抬起到软腭，发 /ŋ/
- **不要发成 /ɪn/！**舌尖不碰上齿龈

---

### **2️⃣ -ng 组（纯 /ŋ/ 结尾）**

| 单词 | 音标 | 元音类型 |
|------|------|----------|
| **long** | /lɔːŋ/ | 长元音 /ɔː/ |
| **wrong** | /rɔːŋ/ | 长元音 /ɔː/ |
| **along** | /əˈlɔːŋ/ | 弱读 /ə/ + 重读 /ɔː/ |
| **strong** | /strɔːŋ/ | 辅音群 str + /ɔːŋ/ |
| **song** | /sɔːŋ/ | 经典 /ɔːŋ/ 组合 |
| **hang** | /hæŋ/ | 短元音 /æ/ |
| **young** | /jʌŋ/ | 短元音 /ʌ/ |
| **darling** | /ˈdɑːrlɪŋ/ | 两个音节，尾音 /ɪŋ/ |

**发音规律：**
- **长元音 /ɔː/ + /ŋ/** 最常见（long, wrong, song, strong, along）
- **短元音 + /ŋ/** 较少（hang, young）
- 元音拖长后，舌根抬起发 /ŋ/

---

### **3️⃣ -nk 组（/ŋk/ 组合）**

| 单词 | 音标 | 分解 |
|------|------|------|
| **think** | /θɪŋk/ | /θ/ + /ɪ/ + **/ŋ/** + **/k/** |
| **thank** | /θæŋk/ | /θ/ + /æ/ + **/ŋ/** + **/k/** |
| **drink** | /drɪŋk/ | /dr/ + /ɪ/ + **/ŋ/** + **/k/** |
| **bank** | /bæŋk/ | /b/ + /æ/ + **/ŋ/** + **/k/** |

**重要！/ŋk/ ≠ /nk/**

| 错误发音 | 正确发音 |
|----------|----------|
| ❌ /n/ + /k/ = 舌尖+软腭 | ✅ /ŋ/ + /k/ = 舌根+软腭 |
| ❌ think /θɪnk/ | ✅ think /θɪŋk/ |

**发音步骤：**
1. 发元音（如 /ɪ/）
2. 舌根抬起到软腭发 **/ŋ/**（鼻音）
3. 保持舌根位置，爆破发 **/k/**
4. 整个过程舌根不离开软腭！

---

### **4️⃣ -le 组（特殊）**

| 单词 | 音标 | 说明 |
|------|------|------|
| **uncle** | /ˈʌŋkl/ | /ŋk/ + 成音节 /l/ |

**发音：**
- /ʌ/ 短元音
- /ŋk/ 组合（舌根位置）
- /l/ 成音节（舌尖弹起）

---

## 🔍 /ŋ/ vs /n/ 对比

| 音标 | 发音位置 | 舌头位置 | 例子 |
|------|----------|----------|------|
| **/n/** | 齿龈 | **舌尖**顶上齿龈 | sin /sɪn/ |
| **/ŋ/** | 软腭 | **舌根**顶软腭 | sing /sɪŋ/ |

**练习对比：**
```
sin /sɪn/  (舌尖) ←→ sing /sɪŋ/  (舌根)
thin /θɪn/ (舌尖) ←→ thing /θɪŋ/ (舌根)
```

---

## 🎯 发音技巧总结

### **1. -ing 结尾 (/ɪŋ/)**
✅ 短促的 /ɪ/，然后舌根抬起  
❌ 不要发成 /ɪn/（舌尖不要碰）

### **2. -ong/-ang/-ung 结尾**
✅ 元音拖长，舌根自然上升  
❌ 不要发得太重，自然收尾

### **3. -nk 结尾 (/ŋk/)**
✅ 舌根一个位置完成两个音  
❌ 不要分成两个动作

---

## 💪 练习建议

### **阶梯训练：**

**Level 1：单独练 /ŋ/**
```
ng... ng... ng... (感受舌根位置)
```

**Level 2：对比练习**
```
sin - sing
thin - thing  
ban - bang
```

**Level 3：-ing 流畅度**
```
sing - ring - bring - thing - morning
```

**Level 4：-nk 难度**
```
think - thank - drink - bank
```

**Level 5：句子**
```
I think something is wrong this morning.
The young king can sing a long song.
```

---

## ⚠️ 常见错误

| 错误 | 原因 | 纠正 |
|------|------|------|
| sing 读成 /sɪn/ | 用了 /n/ 而不是 /ŋ/ | 舌根往后，不是舌尖 |
| think 读成 /θɪnk/ | 把 n 和 k 分开了 | 舌根一次完成 /ŋk/ |
| morning 读成 /mɔːnɪn/ | 最后用了 /n/ | 最后是 /ŋ/，舌根抬起 |
