## 📝 总结提示

- **/tʃ/ 和 /dʒ/** = 爆破音（先憋气再爆破）
- **/ʃ/ 和 /ʒ/** = 摩擦音（持续气流摩擦）
- **清辅音** (/tʃ/, /ʃ/) = 声带不震动
- **浊辅音** (/dʒ/, /ʒ/) = 声带震动 ⭐
