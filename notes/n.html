📋 按 n 位置分组

1️⃣ n 在词首（爆发式，向外推）

单词音标说明
no/noʊ/n 起始，有力推出
not/nɑːt/n 起始，清晰爆发
now/naʊ/n 起始，向外释放
need/niːd/n 起始，长元音跟随
never/ˈnevər/n 起始，开启整个单词

2️⃣ n 在词尾（收敛式，往回收）

单词音标说明
in/ɪn/n 收尾，舌尖停住
on/ɑːn/n 收尾，鼻音收住
can/kæn/n 收尾，柔和结束
one/wʌn/n 收尾，鼻音尾
when/wen/n 收尾，收敛
an/æn/n 收尾，短促
then/ðen/n 收尾，齿音后收
been/bɪn/n 收尾，双唇后收
man/mæn/n 收尾，鼻音尾
mean/miːn/n 收尾，长元音后收
down/daʊn/n 收尾，双元音后收

3️⃣ n 在词中（连接作用）

单词音标说明
know/noʊ/n 在开头位置
want/wɑːnt/n 在中间，连接元音和 t
only/ˈoʊnli/n 在中间，连接两个音节
and/ænd/n 在中间，连接元音和辅音

🎯 总结对比

位置数量发音特点气流方向
词首 n-5个爆发、清晰、有力向外推
词尾 -n11个收敛、柔和、停住往回收
词中 -n-4个连接、过渡、中性中间过渡

💡 练习建议

分组练习法：

1. 先练词首 n（爆发感）：

no - not - now - need - never

2. 再练词尾 n（收敛感）：

in - on - can - when - an - then - been - man - mean - down

3. 最后练词中 n（连接感）：

and - want - know - only

每组重复 20 遍，让嘴巴记住不同位置的 n 的肌肉记忆！💪
