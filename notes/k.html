/k/ 音标练习单词列表

can, like, come, think, look, take, okay, back, make, thank, could, talk, call, work, because, ask, kill, keep, kid, kind

🎯 /k/ 发音说明

发音方法：

  1. 舌位： 舌后部抬起，抵住软腭（口腔后上方）
  2. 嘴型： 嘴巴自然张开
  3. 气流： 气流在口腔后部积聚，舌后部突然离开软腭，气流爆破而出
  4. 声带： 不震动（清辅音）
  5. 口诀： 舌后部抵住软腭，发出"咔"的爆破声

📍 /k/ 拼写规律

拼写形式例词
c- (词首，后接 a/o/u)can, come, could, call
k- (词首，后接 i/e)kill, keep, kid, kind
-k (词尾)think, look, take, okay, back, make, thank, talk, work, ask
-ck (词尾)back
-c- (词中)because
li-ke (特殊)like

🔄 常见对比

音标特点例词提示
/k/清辅音，不震动，爆破音can, back舌后部抵软腭，声带不震动
/g/浊辅音，震动，爆破音go, bag舌后部抵软腭，声带震动
/t/清辅音，齿龈爆破音tan舌尖抵齿龈
/h/清辅音，喉擦音hand气流从喉咙出，不受阻

对比练习：

  • /k/ vs /g/: came → game, class → glass, back → bag, lock → log
  • /k/ vs /t/: came → tame, care → tear, back → bat
  • /k/ vs /h/: came → ham, kind → hind

🗣️ 练习技巧

✅ 记忆口诀： 舌后部"咔"的爆破声 = /k/
✅ 检查方法：

  • 把手放在嘴前，能感觉到强烈的气流喷出
  • 声带不震动（把手放喉咙上检查）
  • 在纸条或蜡烛前发音，能看到纸条/火焰晃动
  • 舌后部能感觉到抵住软腭 ✅ 常见错误：
  • ❌ 声带震动（发成 /g/）
  • ❌ 舌尖抵齿龈（发成 /t/）
  • ❌ 气流太弱（爆破不明显）
  • ❌ 舌位太靠前

💡 重要提示

⭐ /k/ 是软腭爆破音，是 /g/ 的清音对应
⭐ 关键特征： 舌后部抵住软腭，气流积聚后爆破，声带不震动
⭐ 拼写规律：

  • c + a/o/u 发 /k/：can, come, could, call
  • c + i/e 通常发 /s/：city, center（不在本列表中）
  • k + i/e 发 /k/：kill, keep, kid, kind
  • 词尾 -k 或 -ck 都发 /k/：think, back, talk, work
  • -ck 只出现在短元音后：back, clock, luck
  • -ke 通常在长元音或双元音后：like, take, make ⭐ 送气特点：
  • 词首的 /k/ 送气明显：can, come, keep（气流很强）
  • 在 /s/ 后的 /k/ 不送气：sky, school（气流较弱）
  • 词尾的 /k/ 轻收音：think, look, back（爆破不完全） ⭐ 特殊情况：
  • talk /tɔːk/, walk /wɔːk/ 中的 l 不发音
  • could /kʊd/ 中的 l 不发音
  • okay /əʊˈkeɪ/ 拼写特殊

📌 拼写规律详解

字母 c 的发音规则：

后接字母发音例词
a, o, u/k/can, come, could, call
i, e, y/s/city, center, cycle

字母 k 的发音：

  • k 几乎总是发 /k/ 音：kill, keep, kid, kind
  • 极少数例外：know, knife, knee 中的 k 不发音

词尾 -k vs -ck：

拼写条件例词
-ck短元音后back, clock, luck, sick
-k长元音/双元音/辅音后think, look, talk, work, ask
-ke长元音/双元音后like, take, make

🔍 发音位置对比

/k/ 发音部位：

  • 舌后部 + 软腭（口腔后上方）
  • 感觉在喉咙深处爆破

对比其他爆破音：

音标发音部位舌位
/p/双唇舌头不参与
/b/双唇舌头不参与
/t/齿龈舌尖
/d/齿龈舌尖
/k/软腭舌后部
/g/软腭舌后部

💡 记忆技巧

C vs K 选择规则：

  • 看后面的元音！
    • a, o, u → 用 c：can, come, could
    • i, e → 用 k：kill, keep, kid, kind

词尾拼写规则：

  • 短促的音 → 用 -ck：back
  • 长长的音 / 多音节 → 用 -k：think, look, talk, work
